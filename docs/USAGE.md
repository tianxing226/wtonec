# Wtonec 使用教程

## 进入语音面板

1. 确认已在 LSPosed 中启用 Wtonec 并勾选微信作用域。
2. 完全结束微信后重新启动。
3. 首次启动等待 DEX 匹配完成。
4. 进入聊天页面，长按微信语音按钮打开 Wtonec。

## 预设音色

1. 打开“预设”页。
2. 在文字输入框中填写需要合成的内容。
3. 搜索或浏览音色列表，选择一个预设音色。
4. 使用试听确认效果，或点击“生成”仅创建音频。
5. 点击“生成并发送”将处理后的语音发送到当前会话。

## 克隆音色

1. 打开“克隆”页。
2. 添加 Fish Audio 音色 ID，例如文档占位值 `VOICE_ID`。
3. 为音色填写便于识别的名称并保存。
4. 从自定义列表选择音色，输入文字后试听、生成或发送。
5. 可保存多个音色 ID，并按需要切换。

## 本地语音包

1. 打开“语音包”页。
2. 点击导入并从系统文件选择器选取 MP3。
3. 在列表中搜索、排序或试听音频。
4. 使用重命名整理常用素材；不再需要的项目可删除。
5. 点击发送，将选中的语音包发送到当前聊天。
6. 使用导出功能保存需要备份或迁移的音频。

## 设置

- 配置和更新自己的 API Key。
- 查看或维护 DEX 缓存状态。
- 管理语音生成和缓存相关选项。
- 排查问题时查看应用提供的状态与日志入口。

## 安全页

安全页当前显示接口框架与开发状态。Hook 环境、模块清单、进程信号、权限、网络和数据访问风险等扫描接口已预留，扫描执行路径仍处于待开发状态，页面不生成评分、模块清单或风险结论。

## 常见检查

- **面板未出现**：确认 LSPosed 作用域、结束微信进程并重新启动，再检查 DEX 匹配状态。
- **生成按钮停用**：先在设置页保存自己的 API Key，并确认文字和音色已选择。
- **接口请求失败**：查看服务方接口状态、账户余额、网络连接和 Key 状态。
- **导入未完成**：重新从系统文件选择器选择可读取的 MP3，并保留选择器所需权限。
