# Wtonec 使用教程

## 打开面板

进入微信聊天页，将输入栏切换到语音模式，长按微信语音按钮。发送前确认面板顶部显示的是目标聊天。

## Fish Audio

1. 打开“预设”，选择 Fish Audio。
2. 输入文字并搜索音色。
3. 选择音色后试听、生成或生成并发送。

“克隆”页可保存多个 Fish Audio 音色 ID 和显示名称。Fish Audio 在线生成需要自己的 Tiax API Key。

## 轻颜免费

1. 在“预设”页切换到“轻颜免费”。
2. 搜索并选择音色。
3. 输入文字后生成、试听、保存或发送。

轻颜在线生成同样需要 Tiax API Key。

## 系统 TTS

1. 在“预设”页切换到“系统 TTS”。
2. 输入文字，可插入语气标签或文字模板。
3. 调整语速和音调。
4. 点击生成、试听、保存或发送。

系统 TTS 使用 Android 当前默认语音引擎，不依赖音色列表、voiceId 或 API Key。引擎不可用时点击“系统语音设置”安装或切换语音引擎及语言数据。

## 本地语音包

- 点击“导入 MP3”选择本地文件。
- 支持试听、搜索、排序、重命名、导出、删除和发送。
- 长按聊天中收到的语音，可使用简短的保存入口加入 Wtonec 语音包。

导入文件需要是可解码音频。发送前 Wtonec 会在本地完成微信语音格式处理。

## 面板风格

在 Wtonec 独立应用“设置”中选择：

- 液态玻璃：采样当前聊天背景。
- Miuix 浅色：强调清晰可读。
- AMOLED 深色：适合深色环境。

## 安全扫描

安全页可扫描当前设备可见的微信作用域、模块包、进程和本地配置证据，并导出本地报告。结果受 Android 权限、Root 状态和 LSPosed 可见性影响，不等同于对第三方模块源码的完整审计。
