# 故障排查

## 模块未激活

1. 确认 LSPosed 中已启用 Wtonec。
2. 确认作用域包含微信 `com.tencent.mm`。
3. 完全结束微信进程后重新启动。

## 长按没有打开面板

- 进入聊天页并切换到语音模式。
- 长按微信语音按钮，不是短按。
- 检查 DEX 必需项目是否成功。
- 查看 Wtonec 日志和 LSPosed 日志。

## DEX 页面重复出现

- 让匹配过程完整结束，不要切后台。
- 匹配成功后完全结束并重启微信。
- 微信版本变化后重新匹配属于正常情况。
- 同时使用其他微信模块时，保留 Wtonec 与对方模块各自的缓存，不要互相复制。

## 在线音色失败

- 确认 Key 已保存。
- 检查网络、接口状态、余额和文字长度。
- Fish 克隆模式检查音色 ID。
- 分享日志前遮盖 Key 和签名下载地址。

## 系统 TTS 失败

- 音色列表为空并不代表系统 TTS 异常，Wtonec 默认使用当前系统语音。
- 打开“系统语音设置”，确认已安装默认引擎和中文语言数据。
- 首次初始化可能较慢，等待完成后再试。
- 切换引擎后重新打开 Wtonec 面板。
- 系统 TTS 不使用 Tiax API Key，网络接口状态不会影响它。

## MP3 导入或语音保存失败

- 确认文件可正常播放且不是损坏或伪装格式。
- 通过 Android 系统文件选择器重新选择。
- 检查存储空间，并避免同时重复导入同一文件。

## 液态玻璃看不清

在 Wtonec 设置中切换为 Miuix 浅色或 AMOLED 深色。默认聊天背景下会使用稳定的材质回退，自定义背景下使用当前聊天背景采样。

## 提交问题时附带

- Wtonec 版本、versionCode 和 APK SHA-256。
- Android、微信和 LSPosed 版本。
- 出错步骤、Wtonec 日志和对应时间段的 LSPosed 日志。

请遮盖 API Key、账号、联系人、会话、输入文字和本地隐私路径。
