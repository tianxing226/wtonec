# 使用说明与责任边界

Wtonec 仅供学习、研究与技术交流使用。使用者应遵守所在地法律法规、软件平台规则及第三方服务条款，并自行承担安装、配置、升级、内容来源和使用行为产生的责任。

Wtonec 是独立维护的 Android 微信/QQ 双宿主语音工具。微信、QQ、腾讯、LSPosed、Fish Audio、Tiax 及其名称和商标归各自权利主体所有；文档中的名称仅用于描述兼容目标、运行环境和接口连接。

本项目与上述平台、框架和服务提供方不存在隶属、合作、官方授权或背书关系。严禁用于侵犯隐私、传播侵权内容、欺诈、骚扰、批量滥用、干扰平台服务或其他违法用途。第三方接口、内容许可、价格和可用性以对应服务方当前规则为准。

使用者应自行确认设备环境、宿主版本、输入文字、音色素材、在线来源、导入音频和生成结果的使用范围，并管理自己的 API Key、余额和调用记录。接口状态、价格、内容处理和数据规则以服务方当前页面为准。

Android、微信、QQ、LSPosed 或模块版本变化都可能影响 Hook、DEX、AIO、Contact、语音气泡和文件访问。升级前建议保留当前 APK、记录 SHA-256、导出重要 MP3 并保存相关日志。

清除 Wtonec、微信或 QQ 数据，卸载相关应用或执行系统存储清理，可能影响对应宿主副本、共享语音库和缓存。重要内容应提前导出。

安全页提供本地证据扫描。扫描结果受 Android 权限、Root 状态和 LSPosed 可见性影响，仅用于设备排查，不代表对第三方模块源码或网络服务的完整审计。

公开仓库中的 Kotlin 文件仅为接口与模型样例，不构成完整模块或 SDK。提交问题报告前，请先移除 API Key、账号、联系人、群号、会话内容、设备标识和其他个人数据。
