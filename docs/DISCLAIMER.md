# 使用说明与责任边界

Wtonec 是独立维护的 Android 微信语音工具。微信、腾讯、LSPosed、Fish Audio、Tiax 及其名称和商标归各自权利主体所有；文档中的名称仅用于描述运行环境、兼容目标和接口连接。

使用者需要自行确认设备环境、微信版本、输入文字、音色素材、导入音频和生成结果的使用范围，并管理自己的 API Key、账户余额和接口调用。接口可用性、价格、内容处理和调用记录由接口服务方当前规则决定。

Android、微信、LSPosed 或模块版本变化都可能改变兼容状态。升级前建议保留当前 APK、记录 SHA-256、导出重要 MP3，并保存相关日志。微信清除数据、卸载微信或系统存储清理可能影响 `/storage/emulated/0/Android/data/com.tencent.mm/Wtonec/`。

安全页提供本地证据扫描。扫描结果受 Android 权限、Root 状态和 LSPosed 可见性影响，只用于设备排查，不等同于对第三方模块源码或网络服务的完整审计。

公开仓库中的 Kotlin 文件仅为接口与模型样例，不构成完整模块或 SDK。提交问题报告前，请先移除 API Key、账号、联系人、会话内容、设备标识和其他个人数据。
