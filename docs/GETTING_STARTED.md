# 安装与首次启动

## 环境准备

- Android 设备已安装兼容的微信版本。
- LSPosed 环境工作正常。
- 已从 [Wtonec Releases](https://github.com/tianxing226/wtonec/releases) 获取 APK。
- 已准备自己的 Tiax API Key，获取步骤见 [API_KEY.md](API_KEY.md)。

## 安装步骤

1. 安装 Wtonec APK。
2. 打开 LSPosed，在模块列表中启用 Wtonec。
3. 进入 Wtonec 的作用域设置并勾选微信。
4. 从系统最近任务中结束微信，确认微信进程完全退出。
5. 重新启动微信，首次运行时等待 DEX 匹配完成。
6. 进入任意私聊或群聊页面，长按微信语音按钮。
7. Wtonec 面板出现后，进入“设置”页填写 API Key 并保存。

## 首次检查

- Wtonec 主页显示模块与语音功能状态。
- DEX 更新流程结束后，重启微信时缓存状态保持有效。
- 长按语音按钮可以打开包含“预设 / 克隆 / 语音包 / 设置”的面板。
- Key 为空时，语音生成功能保持停用；保存有效 Key 后再进行生成。

## 更新版本

安装新 APK 后，完全结束并重新启动微信。若新版本要求更新 DEX 缓存，等待匹配流程结束后再进入聊天页面使用语音面板。
