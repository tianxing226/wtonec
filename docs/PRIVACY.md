# 隐私说明

## 在线语音

使用 Fish Audio、轻颜或克隆音色时，输入文字、音色参数和 API Key 会发送到 Tiax 接口。接口的处理、计费和保留规则以服务方当前说明为准。

## 本地语音

Android 系统 TTS 与本地语音包不向 Tiax 发送文字或 API Key。系统 TTS 由设备当前语音引擎处理；该引擎自身的网络行为取决于引擎提供方。

## Key 与文件

- API Key 使用 Android Keystore 支持的 AES/GCM 加密保存。
- 语音包、DEX 缓存、日志和崩溃信息保存在微信应用数据目录下。
- 安全扫描读取当前设备可见的本地证据，扫描报告只写入用户选择的本地文件。
- Wtonec 不自动上传安全扫描报告、日志或本地语音包。

## 分享诊断材料

发送截图或日志前，请遮盖：

- API Key、Token 和签名 URL。
- 微信账号、联系人和会话标识。
- 输入文字、音色 ID、文件路径和设备标识。

清除应用数据或卸载相关应用前，先导出需要保留的 MP3。
