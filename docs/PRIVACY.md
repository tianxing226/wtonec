# 隐私说明

## 在线语音请求

使用 Fish Audio、轻颜或克隆音色时，输入文字、音色参数和 API Key 会发送给 Tiax 接口服务。接口处理、计费、调用记录和保留规则以服务方当前页面为准。

在线语音目录可能访问 Wtonec 目录、铃声多多、爱给网或条目声明的媒体地址。来源可用性和数据处理规则由对应站点决定。

## 本地处理

- Android 系统 TTS 不向 Tiax 发送文字或 Key；语音引擎本身的联网行为由引擎提供方决定。
- 本地语音包的解码、重采样和 Tencent SILK 编码在设备本地完成。
- 微信与 QQ 的发送分别使用当前宿主进程中的会话和消息接口。
- 当前聊天背景采样只用于面板材质计算，原始聊天截图不写入磁盘。

## Key、Bridge 与文件

- API Key 由模块 UID 的 Android Keystore 和 AES/GCM canonical store 加密保存。
- 微信与 QQ 通过 UID、包名和签名检查后的 Bridge 访问共享配置。
- 每个宿主保留自己的 DEX 缓存、日志、崩溃信息和语音文件副本。
- canonical 语音库由模块 UID 持有，通过 PFD/Binder 与宿主同步。
- Wtonec 不自动上传安全扫描报告、日志或本地语音包。

## 日志与诊断

日志可能包含宿主版本、功能阶段、错误分类、文件大小、耗时和脱敏标识。QQ 发送日志使用匿名 request 前缀，不记录明文 UIN、UID、会话文本或 API Key。

## 分享诊断材料

发送截图或日志前请遮盖：

- API Key、Token 和签名 URL。
- 微信/QQ 账号、联系人、群号、UID/UIN 和会话标识。
- 输入文字、音色 ID、文件路径和设备标识。

清除 Wtonec、微信或 QQ 数据以及卸载相关应用前，先导出需要保留的 MP3。
