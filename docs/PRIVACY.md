# 隐私说明

## 在线请求

使用文字转语音时，Wtonec 会向 Tiax 接口服务提交：

- 输入文字 `msg`；
- 预设音色编号 `ys`，或 Fish Audio 音色 ID `kl`；
- 使用者配置的 `apikey`。

接口服务方负责请求处理、计费、调用记录和服务日志。具体规则以 [实时接口详情](https://www.tiax.pw/api_details.php?id=7) 和登录后的账户协议为准。

接口返回 MP3 后，Wtonec 在设备本地完成时长检查、微信语音数据生成、试听、缓存、保存和发送准备。

## API Key

API Key 使用 Android Keystore 支持的 AES/GCM 加密后保存。外部数据目录、运行日志、崩溃日志、语音索引、公开文档和公开样例中不记录 Key 明文。

界面展示 Key 时只显示遮盖后的首尾字符。截图或录屏前仍应检查输入框和系统自动填充提示。

## 本地数据

标准微信外部数据根目录为：

```text
/storage/emulated/0/Android/data/com.tencent.mm/Wtonec/
```

本地数据可能包含：

| 数据类型 | 可能内容 |
|---|---|
| 已保存语音 | MP3、微信语音数据、时长和 SHA-256 |
| 音色与索引 | 显示名称、Fish Audio 音色 ID、备注、文本预览和绝对路径 |
| DEX 缓存 | 微信版本指纹、匹配描述符、有效状态和更新时间 |
| 运行日志 | 功能阶段、错误信息、时间、微信版本和本地路径 |
| 崩溃诊断 | Java/native 堆栈、设备与构建状态、错误上下文 |
| 任务缓存 | 生成中的 MP3、微信语音数据和临时文件 |

完整位置见 [STORAGE.md](STORAGE.md)。

## 保存周期与清理

- 运行日志按日期生成，代码中的自动轮转会清理较早日志。
- 崩溃目录保留有限数量的最新诊断文件。
- 语音任务缓存受容量和时间策略管理，也可从设置页清理。
- 已保存语音包持续保留，直到使用者在语音包页面删除或清理微信数据。
- DEX 缓存在微信版本指纹变化或手动清理后重新生成。

## 分享诊断材料

提交问题前检查并遮盖：

- API Key 与账户信息；
- 联系人、群聊和会话标识；
- 输入文字、语音内容和音色 ID；
- 本地绝对路径、设备标识和网络信息；
- 与问题无关的日志区段。

优先导出单个相关日志，而不是打包整个 Wtonec 目录。
