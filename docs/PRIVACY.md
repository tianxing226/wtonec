# 隐私说明

## 在线请求

使用文字转语音时，输入文字、预设音色参数或 Fish Audio 音色 ID，以及用户配置的 API Key 会发送到 Tiax 接口服务。接口服务方对请求处理、调用记录、计费和服务日志的规则以其当前页面与账户协议为准。

## API Key

Wtonec 在应用内保存用户配置的 Key，并仅在调用语音接口时使用。公开仓库、文档、截图和 Kotlin 样例使用 `YOUR_API_KEY` 等占位符，不包含调试 Key 或用户 Key。

## 本地数据

Wtonec 会在设备本地维护功能所需的数据，例如：

- 音色目录与自定义音色配置；
- 生成和导入的音频缓存；
- 本地语音包元数据；
- DEX 匹配缓存与应用设置；
- 诊断所需的本地日志。

清理缓存、删除语音包或清理日志前，请先导出仍需保留的音频。

## 安全页状态

安全页当前为扩展框架和状态展示。扫描执行逻辑仍在开发计划中，当前版本不据此生成其他模块清单、安全评分或风险判断。

## 反馈材料

提交问题时，建议附上版本号、复现步骤和经过遮盖的日志。请先移除 API Key、账号信息、会话标识、联系人内容和本地文件路径。
