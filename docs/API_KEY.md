# API Key 获取与配置

Wtonec 的在线文字转语音功能使用 Tiax 提供的 Fish Audio 语音接口。每位使用者都需要配置自己的 API Key。

## 官方入口

- [Tiax 用户中心](https://www.tiax.pw/user/)
- [Fish Audio 文字转语音接口详情](https://www.tiax.pw/api_details.php?id=7)

接口状态、音色数量、单次价格、余额规则和调用限制会随服务方配置变化，以实时页面和登录后的控制台为准。

## 获取流程

1. 打开 Tiax 用户中心。
2. 已有账号直接登录；新账号从登录页面进入注册流程。
3. 登录控制台后进入 API 密钥管理区域。
4. 按控制台提示获取并管理 Key。
5. 查看账户余额、调用记录和当前计费说明。

## 在 Wtonec 中保存

1. 打开微信聊天页面。
2. 长按微信语音按钮进入 Wtonec。
3. 切换到“设置”。
4. 在 `Tiax API Key` 输入框填写自己的 Key。
5. 点击保存，等待保存成功提示。

Wtonec 使用 Android Keystore 支持的 AES/GCM 加密保护 Key，并将密文保存在应用配置中。Key 不写入 `/storage/emulated/0/Android/data/com.tencent.mm/Wtonec/`，也不写入公开仓库、日志和语音元数据。

## 请求参数

语音生成会使用以下参数：

| 参数 | 用途 |
|---|---|
| `msg` | 待合成文字 |
| `ys` | 预设音色编号 |
| `kl` | Fish Audio 自定义音色 ID |
| `apikey` | 使用者配置的 API Key |

`ys` 与 `kl` 按模式二选一。接口成功响应为 MP3，后续语音处理在设备本地执行。

公开示例统一使用：

```text
YOUR_API_KEY
VOICE_ID
TEXT
```

## Key 管理

- 每个账号使用自己的 Key。
- 截图、录屏、日志和问题反馈中遮盖 Key。
- 定期查看服务方控制台中的调用记录和余额。
- Key 状态异常时，在服务方控制台更新密钥，并同步更新 Wtonec 设置。
- 清除 Wtonec Key 后，在线生成保持停用，已保存的本地语音包仍保留在文件目录中。

## 常见检查

- **保存后仍提示未配置**：重新打开设置页核对遮盖后的 Key 状态，并完整重启微信。
- **HTTP 请求失败**：检查网络、接口实时状态、余额和调用记录。
- **预设可选但生成失败**：确认文字长度、音色参数和 Key 均有效。
- **克隆音色失败**：核对 Fish Audio 音色 ID，避免复制空格或说明文字。
