# API Key 获取与配置

Wtonec 的在线文字转语音功能使用 Tiax 提供的 Fish Audio 语音接口。每位用户都需要配置自己的 API Key。

## 获取 Key

1. 打开 [Tiax 用户中心](https://www.tiax.pw/user/)。
2. 已有账号可直接登录；新用户可从登录页进入注册页面。
3. 登录控制台后，在 API 密钥管理区域获取并管理自己的 Key，同时可查看调用记录。
4. 账户余额、充值或购买方式以登录后的控制台提示为准。

服务方公开页面当前显示 Fish Audio 文字转语音接口采用按次计费。价格、接口状态、音色数量和调用规则都可能调整，请在使用前查看 [接口实时详情](https://www.tiax.pw/api_details.php?id=7)。

## 在 Wtonec 中配置

1. 打开微信聊天页面。
2. 长按微信语音按钮进入 Wtonec。
3. 切换到“设置”。
4. 在 `Tiax API Key` 输入框中填写自己的 Key。
5. 点击保存，等待页面提示保存完成。

Key 为空时，语音合成功能保持停用。文档和公开样例统一使用 `YOUR_API_KEY`，仓库中不保存真实 Key。

## 请求中包含的数据

生成语音时，Wtonec 会向接口服务方提交：

- 输入文字 `msg`；
- 预设音色参数 `ys`，或 Fish Audio 音色 ID 参数 `kl`；
- 用户配置的 `apikey`。

`ys` 和 `kl` 按生成模式二选一。接口返回 MP3 音频后，应用继续完成本地处理。

## Key 管理建议

- 每位用户使用自己的 Key，不共享到账户外部。
- 截图、日志和问题反馈中对 Key 做遮盖处理。
- 怀疑 Key 已泄露时，进入服务方控制台更新密钥，并同步更新 Wtonec 配置。
- 关注调用记录与账户余额，价格与状态以服务方实时页面为准。
