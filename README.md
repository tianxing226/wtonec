# Wtonec

Wtonec 是一个 Android 微信语音工作台，提供文字转语音、Fish Audio 自定义音色、本地语音包和微信兼容 SILK 音频发送流程。

## 功能介绍页

[https://tianxing226.github.io/wtonec/](https://tianxing226.github.io/wtonec/)

页面源码位于仓库根目录：

- `index.html`
- `styles.css`
- `app.js`
- `assets/`

推送到 `main` 后，`.github/workflows/pages.yml` 会构建并发布 GitHub Pages。
